## Шаг 2 — Реальная информация

Обновить в `index.html`:
- Имя невесты
- Дата и время события
- Имена хозяев (родители)
- Адрес и название ресторана
- Город
- Ссылка на карту (Google Maps / 2GIS)
- Текст приглашения (казахский / русский)
- Язык по умолчанию (KZ или RU)

## Шаг 4 — RSVP функционал

### Вариант A — Google Sheets (рекомендован)
- Форма отправляет данные через **Google Apps Script** (бесплатно, без сервера)
- Ответы пишутся в Google Таблицу: имя, вариант ответа, дата/время
- Нужен Google аккаунт для настройки Apps Script

### Вариант B — Airtable
- REST API, красивый интерфейс таблицы
- Бесплатный тариф ограничен 1000 записей
- Нужен API ключ

### Вариант C — Notion
- Через Notion API
- Сложнее в настройке

**Решение по варианту:** ✅ Google Sheets (Apps Script)

### Поля для сохранения:
- Имя гостя
- Вариант ответа (Қатысамын / Жұбайыммен / Келе алмаймын)
- Дата и время отправки
- Язык интерфейса (KZ/RU)
